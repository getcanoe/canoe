Soft Lock

With Soft Lock, BCB wallet is locked but your wallet is still live unencrypted in memory. Enabling Soft Locks, makes it possible to use simpler forms of credentials like PINs and fingerprints.

Hard Lock
With Hard Lock, BCB wallet encrypts your wallet and completely remove it from memory. You can not disable Hard Lock, but you can set a very high timeout.
Background Behaviour

Choose the lock type to use when BCB wallet goes to the background. To disable background locking select None.
